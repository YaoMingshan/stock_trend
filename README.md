# 📈 A股趋势分析系统

自动化A股涨跌排行分析系统，每日自动更新，通过GitHub Pages展示。

## ✨ 功能特性

- 📊 **多周期分析**: 5日、10日、20日涨跌排行
- 🏆 **TOP10排行**: 涨幅/跌幅前10名股票
- 📈 **市场概况**: 涨跌家数、涨停跌停统计
- 🤖 **自动更新**: 每个交易日15:30自动运行
- 🌐 **在线展示**: GitHub Pages静态网站

## 🚀 快速开始

### 1. Fork本仓库

点击右上角的 `Fork` 按钮

### 2. 启用GitHub Pages

1. 进入仓库 `Settings` → `Pages`
2. Source选择 `GitHub Actions`

### 3. 启用GitHub Actions

1. 进入 `Actions` 标签页
2. 点击 `I understand my workflows, go ahead and enable them`

### 4. 手动触发首次运行

1. 进入 `Actions` → `每日股票分析更新`
2. 点击 `Run workflow`
